# hanbat-krafton-web-Chatbob

웹 기반 챗봇 애플리케이션입니다.  
크래프톤 연계 2주 웹개발 실무 특강 과제로 제작한 프로젝트입니다.

---

## 프로젝트 소개

브라우저에서 동작하는 간단한 챗봇 웹 서비스입니다.  
사용자의 입력을 받아 화면에 대화 형태로 출력하며, 기본적인 웹 구조와 서버–클라이언트 흐름을 학습하는 것을 목표로 합니다.

---

## 주요 기능

- 챗봇 UI (입력창 및 대화 내역 표시)
- 사용자 메시지 처리 및 응답 출력
- Flask 기반 서버 구조
- 정적 파일 및 템플릿 분리 구조

---

## 기술 스택

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Python, Flask  
- **Template Engine**: Jinja2
